# TamentaiProvider

Esse provider fornece os recursos necessários para que uma série de componentes da biblioteca funcionem adequadamente, funcionando como um agregador de outros providers.

Dentro dele estão os providers a seguir:

### Exemplo de uso
Envolva toda sua aplicação com o provedor do Tamentai para ter acesso a todos os recursos.

```tsx
import React from "react";
import ReactDOM from "react-dom/client";

import App from "./App";
import { TamentaiProvider } from "@poliedro/tamentai";

const root = ReactDOM.createRoot(
  document.getElementById("root") as HTMLElement
);

root.render(
  <React.StrictMode>
    <TamentaiProvider>
      <App />
    </TamentaiProvider>
  </React.StrictMode>
);
```
